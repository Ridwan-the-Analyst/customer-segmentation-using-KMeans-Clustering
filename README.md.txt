# 📊 Customer Segmentation App

This is a **Streamlit** web application that performs **customer segmentation** using **K-Means clustering**. The app allows users to upload a dataset, select numerical features, and visualize segmented clusters.

---

## 🚀 Features
✅ Upload a CSV file  
✅ Select features for clustering  
✅ Determine the optimal number of clusters (Elbow Method)  
✅ Visualize customer segments  
✅ Download the clustered dataset
